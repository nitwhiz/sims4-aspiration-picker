# sims4-aspiration-picker

## Project setup

```shell
pnpm install
```


### Prepare assets

```shell
node scripts/csv2json.js CSV-FILE
```

### Compiles and hot-reloads for development

```shell
pnpm run serve
```

### Compiles and minifies for production

```shell
pnpm run build
```

### Lints and fixes files

```shell
pnpm run lint
```
